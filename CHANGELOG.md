# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project follows Semantic Versioning.

## [Unreleased]

### Added

- GitHub Pages landing pages in Chinese and English for promotion and onboarding.
- Release workflow that builds the production manifest and uploads release assets on version tags.
- Issue templates for bug reports and feature requests.
- One-click sideload installer scripts for macOS and Windows shared-folder setup.

### Changed

- README files now lead with the in-Word revision workflow, target audiences, and faster install path.

## [1.0.0] - 2026-03-14

### Added

- Word task pane chat for selection-aware writing assistance.
- Plan mode for outline generation and section-by-section drafting.
- Context menu commands for polish, translate, comments, and structure analysis.
- Track Changes-aware write-back flow for Word revisions.
- OpenAI-compatible API configuration with local settings persistence.
