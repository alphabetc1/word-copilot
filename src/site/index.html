在 Word 里直接改稿，而不是来回复制粘贴

Word Copilot 把 AI 写作能力直接放进 Microsoft Word。 你可以选中文本后润色、翻译、加批注、做结构检查，并用 Word 修订模式保留每一处改动。

适合论文、临床报告、项目申报、公文等长文场景。配置自己的 API Key 即可使用，不依赖单独账号体系。

Quick Start

30 秒上手

如果你已经有可用的 API Key，实际安装过程只需要一次 manifest 上传和一次模型配置。

下载 Manifest 打开 `word-copilot.xml` 并保存到本地。
加载到 Word 在 Word 中依次点击“插入 → 加载项 → 我的加载项 → 上传我的加载项”。
配置模型 在插件设置中填写 `Base URL`、`API Key`、`Model` 并保存。
开始改稿 选中文本后使用右键菜单，或直接在侧边栏输入指令。
