Microsoft Word Add-in · AI 改稿工作流

在 Word 里直接改稿，不用来回复制粘贴

选中文本即可润色、翻译、加批注、做结构检查，所有改动通过 Word 修订模式保留。 配置自己的 API Key，即装即用。

🔄 修订写回

AI 改写通过 Track Changes 写入文档，可逐条接受或拒绝，审阅协作更安全。

⚡ 右键即用

润色、翻译、批注、结构分析，选中文本右键直接触发，不需要切窗口。

📊 长文友好

计划模式先出大纲再逐段写，结构检查能指出缺失章节和层级问题。

30 秒上手

下载 Manifest 保存 word-copilot.xml 到本地
加载到 Word 插入 → 加载项 → 我的加载项 → 上传我的加载项
配置模型 填写 Base URL、API Key、Model 并保存
开始改稿 选中文本右键，或在侧边栏输入指令

支持 OpenAI / Azure OpenAI / 通义千问及其他兼容接口 · Apache 2.0 开源协议
