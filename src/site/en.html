Quick Start

Install in four steps

If you already have an API key, the real setup is just one manifest upload and one model configuration.

  1. Download `word-copilot.xml`.
  2. In Word, open Insert → Add-ins → My Add-ins → Upload My Add-in.
  3. Enter `Base URL`, `API Key`, and `Model` in settings.
  4. Start with the sidebar or the right-click command menu.
