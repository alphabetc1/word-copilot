Microsoft Word Add-in · AI Revision Workflow

Edit inside Word, not in a separate chat window

Select text to polish, translate, annotate, or check structure. Every change goes through Track Changes so nothing is lost. Bring your own API key and start right away.

🔄 Track Changes

AI edits are written back via Track Changes. Accept or reject each revision individually.

⚡ Right-Click Commands

Polish, translate, annotate, and analyze structure from the context menu. No window switching.

📊 Long-Form Friendly

Plan mode generates outlines first, then drafts section by section. Structure check spots missing parts.

Install in four steps

Download Manifest Save word-copilot.xml to your machine
Load into Word Insert → Add-ins → My Add-ins → Upload My Add-in
Configure Model Enter Base URL, API Key, and Model in settings
Start Editing Right-click selected text or use the sidebar

Works with OpenAI / Azure OpenAI / Qwen and compatible endpoints · Apache 2.0
